HumanTimer

This package can be used as a small dependency to show real time clock for each of your time units. It carefully make use of setTimeout and SetInterval to update the clock.

Optionally you can use rxjs to automatically update the UI as I am doing in this example.

Frameworks

You can set result property of the TimeCalculator into your state which will automatically update the ui.
